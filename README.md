# zenvim

A simple and chill neovim configuration.

## notable features + plugins

- Consistent, transparent colorscheme
- Autocommands for [chezmoi](https://www.chezmoi.io/) (deletable in [autocmds.lua](lua/autocmds.lua))
- LSP support ([mason](https://github.com/williamboman/mason.nvim) + [lspconfig](https://github.com/neovim/nvim-lspconfig) + [mason-lspconfig](https://github.com/williamboman/mason-lspconfig.nvim))
- File finder + explorer ([snacks](https://github.com/folke/snacks.nvim))
- Tabs ([barbar](https://github.com/romgrk/barbar.nvim))
